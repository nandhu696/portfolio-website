# Portfolio Website

This is a personal portfolio website built using Flask (Python web framework).

## 📌 Features
- Home page with personal introduction
- About section
- Projects showcase
- Contact form
- SQLite database integration

## 🛠️ Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript
- SQLite

## 🚀 How to Run This Project Locally

1. Clone the repository:
   git clone https://github.com/nandhu696/portfolio-website.git

2. Go to project folder:
   cd portfolio-website

3. Install Flask:
   pip install flask

4. Run the application:
   python app.py

5. Open browser:
   http://127.0.0.1:5000

## 📁 Project Structure
- app.py → Main Flask application
- templates/ → HTML files
- static/ → CSS, JS, images
- portfolio.db → Database file
